


# 🚀千牛客服机器人

专为淘宝千牛打造的AI客服机器人，实现千牛7×24小时自动化值守，接入deepseek，阿里通义千问AI大模型



1. 仅支持千牛多账号模式
2. 需要打开千牛无障碍模式，如下图

<div align="center">
  <img src="https://worklink.oss-cn-hangzhou.aliyuncs.com/B78EE5CC4F478BE414D95C1CC12E20E2.png" >
  <br>
</div>

### 业务功能
1. 发图文消息 
2. 自动发货
3. 催付卡片 
4. 核对卡片 
5. 转接到个人 
6. 转接到分组
